Glossary

Character reference
A character reference is a code that represents a character
Absolute URLs
The complete URL, including protocol (eg. http) and domain name
Relative URLs
The path relative to the current web page (or CSS or JavaScript file)
HTML attributes
Contain information about the element beyond the content and can be used to identify an element for styling and in scripts
Absolute file paths
Start from disc root
